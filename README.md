# FlowState

FlowState is a dynamic, mood-driven productivity platform that helps users discover personalized content based on their current emotional state. It features integrated Pomodoro timers, curated music, movie, and game recommendations.

## Features
- Mood-based content discovery (Focus, Relax, Energize, Creative, Melancholy)
- Integrated Pomodoro timer for focused work sessions
- Curated playlists from Deezer, movie recommendations from TMDB, and games from Itch.io
- Responsive, modern design
- Accessible navigation and skip links
- Animated UI with GSAP


## Credits
